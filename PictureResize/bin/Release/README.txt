No installation needed, simply put the folder pictureresize somewhere. But .NET 3.5 is needed (download from microsoft if not already on the computer)

Short instructions: making images smaller (in size and filesize)
1. Click on [Load] and select one or more image files to be resized and then [Open]. Or open e.g. explorer and pull the image file(s) in by drag and drop.
2. Choose the hight or width to which your image shall be resized
3. Put in a filename extension for the resized image, this will be added to the original filename to distinguish it
eg. extension -sm will create DSC_1058-sm.jpg from DSC_1058.jpg
4. Choose the desired JPG quality (the higher the quality the nicer the image but the bigger the filesize)
5. You can choose an interpolation mode, in doubt use the default "Bicubic"
6. Now you are ready to resize your images pressing [Los!] (which means [Go!]). As a result the resized images will be stored in the same folder as the orginals

Kurzanleitung: Bilder verkleinern (in Grösse und Speicherbedarf)
1. Klicken sie auf [Bilder laden...] und wählen sie die Bilder aus, die sie verkleinern möchten. Klicken sie auf [Öffnen]. Sie können auch Bilder, die sie z.B. im Explorer ausgewählt haben, per Drag 'n Drop auf die Bilderliste ziehen.
2. Geben sie eine Breite oder Höhe ein, auf die ihre Bilder verkleinert werden sollen.
3. Geben sie eine Dateinamenserweiterung an. Diese wird an die verkleineten Bilder angehängt, damit sie sie von den Originalen unterscheiden können. Beispiel: die Datinamenserweiterung ist "_kl". Aus DSC_1058.jpg wird also DSC_1058_kl.jpg
4. Stellen sie die gewünschte JPG-Qualität ein. Je grösser die Qualität, desto schöner das Bild aber auch desto grösser die Datei.
5. Wählen sie einen Interpolationsmodus aus. Falls sie nicht wissen, was das ist, nehmen sie Bikubisch, dieses Erziehlt meist gute Ergebnisse.
6. Nun können ihre Bilder verkleinert werden. Klicken sie auf [Los!]. Als Resultat sind die verkleinerten Bilder mit dem Namensanhang im gleichen Ordner wie die Originale gepeichert
