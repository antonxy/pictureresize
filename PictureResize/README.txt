Kurzanleitung: Bilder verkleinern
1. Klicken sie auf [Bilder laden...] und wählen sie die Bilder aus, die sie verkleinern möchten. Klicken sie auf [Öffnen]. Sie können auch Bilder, die sie z.B. im Explorer ausgewählt haben, per Drag 'n Drop auf die Bilderliste ziehen.
2. Geben sie eine Breite und/oder Höhe ein, auf die ihre Bilder verkleinert werden sollen.
3. Geben sie eine Dateinamenserweiterung an. Diese wird an die verkleineten Bilder angehängt, damit sie sie von den Originalen unterscheiden können. Beispiel: die Datinamenserweiterung ist "_kl". Aus DSC_1058.jpg wird also DSC_1058_kl.jpg
4. Stellen sie die gewünschte JPG-Qualität ein. Je grösser die Qualität, desto schöner das Bild aber auch desto grösser die Datei.
5. Wählen sie einen Interpolationsmodus aus. Falls sie nicht wissen, was das ist, nehmen sie Bikubisch, dieses Erziehlt meist gute Ergebnisse.
6. Nun können ihre Bilder verkleinert werden. Klicken sie auf [Los!].
